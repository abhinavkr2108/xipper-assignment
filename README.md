## Run Locally

Clone the project

```bash
  git clone https://github.com/abhinavkr2108/xipper-assignment
```

Go to the project directory

```bash
  cd xipper
```

Install dependencies (Backend)

```bash
  npm install
```

Install dependencies (Frontend)

```bash
  cd frontend
```

```bash
  npm install
```

Start the backend server

```bash
  npm run dev
```

Start the frontend server

```bash
  cd frontend
```

```bash
  npm run dev
```
